       Transient Pressure Simulation in Fractured Reservoirs
Authors: Yin, R. & Zhang, S.

Note: Figures marked [OLE] are embedded as Microsoft Office OLE objects
(typically Excel charts or Word drawings) in the original .docx file.
These cannot be directly extracted as standalone image files.
To export them, open the original .docx in Microsoft Word, right-click
each figure, and select "Save as Picture."

All extractable image files are saved in the 'media/' subfolder.
